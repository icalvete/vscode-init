## Comandos pendientes

| Comando | Descripción |
|---------|-------------|
| `/test` | Genera tests para código seleccionado. Detecta el framework (RSpec, pytest, Jest) y crea tests unitarios con casos edge. |
| `/refactor` | Refactoriza código manteniendo el comportamiento. Extrae métodos, simplifica condicionales, aplica patrones. |
| `/fix` | Arregla un error específico. Recibe el error (pegado o de terminal) y propone solución. |
| `/optimize` | Analiza y mejora rendimiento. Busca N+1 queries, loops ineficientes, memoria. |
| `/migrate` | Ayuda a migrar código entre versiones o frameworks. Ej: Rails 6 a 7, Vue 2 a 3. |
