---
description: Auditoría de seguridad del código
---

Buscar vulnerabilidades:
- Inyecciones: SQL, Command, XSS, Path Traversal
- Auth: contraseñas débiles, tokens predecibles, permisos
- Secrets: API keys en código, logs sensibles
- Config: debug en prod, CORS permisivo, HTTPS

Clasificar: CRÍTICO > ALTO > MEDIO > BAJO

Output: resumen, vulnerabilidades con severidad, fix recomendado.
