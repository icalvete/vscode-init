
## JavaScript

### Convenciones
- Variables/funciones: `camelCase`
- Clases/componentes: `PascalCase`
- Constantes: `SCREAMING_SNAKE_CASE`
- Booleanos: prefijos `is`, `has`, `can`
- Indentación: 2 espacios, máx 100 chars

### Idioms
- Destructuring: `const { name } = user`
- Spread: `[...array, item]`
- `async/await` en vez de `.then()`

### Docs
Usar JSDoc: `@param`, `@returns`, `@throws`, `@example`
