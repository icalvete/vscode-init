
## Python

### Convenciones (PEP 8)
- Módulos/funciones/variables: `snake_case`
- Clases: `PascalCase`
- Constantes: `SCREAMING_SNAKE_CASE`
- Privados: prefijo `_`
- Indentación: 4 espacios, máx 99 chars
- Type hints obligatorios

### Idioms
- List comprehensions en vez de `filter/map`
- `with open()` para archivos
- f-strings para formateo

### Docs
Usar Google-style docstrings: `Args:`, `Returns:`, `Raises:`, `Example:`
