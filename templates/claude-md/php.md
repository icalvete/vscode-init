
## PHP

### Convenciones (PSR-12)
- Clases/Namespaces: `PascalCase`
- Métodos/funciones/variables: `camelCase`
- Constantes: `SCREAMING_SNAKE_CASE`
- Indentación: 4 espacios, máx 120 chars
- Type hints obligatorios

### Idioms
- Null coalescing: `$x ?? 'default'`
- Nullsafe: `$user?->getName()`
- `match()` en vez de `switch`

### Docs
Usar PHPDoc: `@param`, `@return`, `@throws`
